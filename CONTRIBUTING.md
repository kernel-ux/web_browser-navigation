# Contributing

Thanks for helping improve GhostGuide.

## Quick Rules

- Keep changes focused and small when possible.
- Prefer clear, readable code over clever code.
- Test your changes in Chrome before submitting.

## Local Testing

1. Load the extension from `chrome://extensions`.
2. Make code changes.
3. Click **Reload** on the extension card.
4. Reopen the side panel and test your flow.

## What to Include in a PR

- What changed and why.
- How you tested it.
- Screenshots or a short clip if UI behavior changed.
